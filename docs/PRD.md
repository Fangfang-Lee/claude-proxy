# Claude Proxy 产品需求文档 (PRD)

## 1. 产品概述

### 产品名称
Claude Proxy

### 产品类型
macOS 桌面应用程序（Apple Silicon）

### 产品定位
一个本地化的 Claude API 代理管理工具，帮助用户轻松管理和切换多个 Claude API 中转服务，解决 API 访问稳定性问题。

### 核心价值
- **多路由支持**：同时管理多个 API 端点，通过规则智能路由请求
- **模型映射**：灵活的模型名转换和别名配置
- **可视化监控**：实时查看请求日志、延迟统计和 Token 用量

---

## 2. 目标用户

1. **开发者**：使用 Claude API 构建应用的开发者
2. **AI 爱好者**：需要稳定访问 Claude 服务的个人用户
3. **企业用户**：有多地区部署需求的技术团队

---

## 3. 功能需求

### 3.1 核心功能

#### F1: 多路由规则管理
- 添加、编辑、删除路由规则
- 配置多个上游 API 端点（URL、API Key）
- 设置规则优先级（拖拽排序）
- 启用/禁用单个规则
- 支持通配符匹配模型名（如 `claude-sonnet-*`）

#### F2: 智能模型映射
- 配置模型别名映射（如 `claude-3-5-sonnet-20241022` -> `sonnet`）
- 默认模型兜底配置
- 支持表单视图和 JSON 视图两种编辑方式

#### F3: 请求日志
- 实时显示请求记录
- 显示匹配规则、响应延迟、状态码
- 查看请求/响应详情（Headers、Body）
- Token 用量统计
- 日志搜索和筛选

#### F4: 速度测试
- 一键测试所有路由的连通性
- 显示各路由的响应延迟
- 标识最优路由

#### F5: Claude Code 集成
- 一键写入 Claude Code 配置
- 支持模型别名配置
- 复制配置到剪贴板

#### F6: 系统托盘
- 托盘图标显示代理状态
- 快速访问应用
- 最小化到托盘

#### F7: 设置管理
- 配置代理端口
- 设置请求超时时间
- 启动时自动运行
- 主题切换（后续版本）

### 3.2 数据存储

| 数据类型 | 存储位置 |
|----------|----------|
| 路由规则 | `~/Library/Application Support/claude-proxy/routes.json` |
| 应用设置 | `~/Library/Application Support/claude-proxy/settings.json` |

---

## 4. 非功能需求

### 4.1 性能要求
- 代理请求延迟 < 50ms（不含上游网络延迟）
- 应用启动时间 < 3 秒
- 日志页面流畅滚动（支持 10000+ 条记录）

### 4.2 兼容性
- macOS 13.0+ (Ventura)
- Apple Silicon (arm64)

### 4.3 安全要求
- API Key 本地加密存储
- 不上传任何用户数据到远程服务器
